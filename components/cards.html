Cards


Design, Uses, and Code:

The card features an image, with a title and a subtitle, and a link to the page it is referencing. The card's primary use is to be a project preview. The image should pertain to the project and feature a screen, and all images on the portfolio site used for card should match in style and presentation. Subtitles should not be more than a few words describing a project.
