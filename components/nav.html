Navigation Bar


Design, Uses, and Code:

The navigation bar (Nav Bar) featured on my portolio site will be a sticky, allowing users to access the navigation system from anywhere on the page. The nav bar will span the width of the entire screen it occupies, with the text buttons condesing into a collapsible menu bar (hamburger) for tablet and mobile screens.

I've left the main bar color white, to simplify the switch for dark mode and the main indications for page selected is through bold and underlined text, so active menu buttons can be easily read.
